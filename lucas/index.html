About Panamax

What?
Panamax is a containerized app creator with an open-source app marketplace hosted in GitHub. Panamax provides a friendly interface for users of Docker, Fleet & CoreOS. With Panamax, you can easily create, share and deploy any containerized app no matter how complex it might be.
Why?
Containers could be the next Virtual Machine. But running multi-container, multi-server apps with containers is very hard. You have to learn 5 new and different technologies and best practices (libswarm, systemd, etcd, ambassadord, fleet, etc) just to get started. Panamax incorporates all the Docker best practices for you and gives you a beautiful UX that does the heavy lifting of deploying complex applications for you.
Where?
You can run Panamax on your laptop or any cloud infrastructure that supports CoreOS (CenturyLink Cloud, Google, Amazon, Rackspace, etc.). You can even run Panamax on bare metal.
Who?
Panamax is the first open-source project to be incubated out of CenturyLink Labs. Our goal was to help developers utilize cutting-edge technologies not by abstracting what they do but by simply providing a friendly interface to them.
How Much?
Free. The Panamax code is open-source and the app template marketplace is open-source. We are committed to building a strong open source community project based on Apache 2 License, you can be certain that we will be good citizens and never build cripple-ware.

How It Works

  • Install

    On a Mac with Homebrew, just run: brew install http://download.panamax.io/installer/brew/panamax.rb && panamax init. For other platforms, see the installation documentation.

  • Build

    Use our beautiful Panamax Web UI to stitch together many Docker containers into anything you can dream up. You can configure links, ports and environment variables, as well as organize your application services into categories.

  • Manage

    Save your application template to a git repository and share it with your friends. You can source your friends' template repositories or submit a pull request and have your template be canonical and accessible by anyone in the world.
